## Tight follow-up before v0.4 kickoff

### [v0.3.6](v0.3.6-plan.md) — small maintenance patch (planned)

Three items, single tag, ~2 hours of work:

1. `macmlx --version` auto-bumped via `release.yml` `sed` step
   (fixes the perpetual "0.1.0" bug)
2. `macmlx search <query>` subcommand (reuses `HFDownloader.search`)
3. Release binary slim-down: `strip -S` + dynamic Swift stdlib (~60
   MB → ~45 MB). Plus CLI `--log-level` + `--log-stderr` flags so
   Pulse logging is actually usable from the command line.

See [`v0.3.6-plan.md`](v0.3.6-plan.md) for the full spec.

### v0.3 cycle sessions + decisions (historical)

- Sandbox policy review → keep on, defer subprocess engines
- Pulse keep / PulseUI remove decision (v0.3.4 → v0.3.5)
- SwiftTUI → in-house ANSI rendering decision (v0.3.5)
- "Chat history management" scope expansion after user request
- VLM + adapter + speech roadmap shaping (see v0.4 / v0.5 / v0.6
  entries in README roadmap)

Archived stage-by-stage v0.1 plans are in
[`archive/`](archive/) for historical reference.
