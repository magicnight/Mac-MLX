# macMLX v0.1 — Master Implementation Overview

> **For agentic workers:** This file is the contract for what gets shipped in v0.1 and the order of work. Each Stage has (or will have) its own detailed task-by-task plan in this directory. Use `superpowers:subagent-driven-development` to execute Stage plans.

**Goal:** Ship the first working version of macMLX — a native macOS LLM inference app with SwiftUI GUI, CLI + TUI, and OpenAI-compatible HTTP API, distributed via DMG with Sparkle auto-update.

**Architecture:** Three-target Swift monorepo. `MacMLXCore` SPM package owns all inference, model management, settings, and HTTP server logic. `macMLX.app` (Xcode project) and `macmlx` (SPM executable) are thin shells that import Core. Engine implementations sit behind a single `InferenceEngine` Swift protocol so the rest of the app is engine-agnostic.

**Tech Stack:** Swift 6.3 strict concurrency · SwiftUI (macOS 14+) · `mlx-swift-lm` (Apple SPM) · `Hummingbird` (HTTP server) · `swift-argument-parser` · `SwiftTUI` · `Sparkle 2.x` · Swift Testing.
